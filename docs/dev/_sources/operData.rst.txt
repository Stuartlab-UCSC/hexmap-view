
Data server commands
====================

Start data server
.................

::

 cd $HEXCALC
 bin/start

Stop data server
................

::

 cd $HEXCALC
 bin/stop

Check data server process
.........................

::

 ps -eaf | grep TBD

Log file
........
The log file is at compute/ops/log.
