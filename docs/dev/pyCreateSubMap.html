Python API: syncMaps¶

This python API is used in communications between the map manager and the agent to feature translator. The full pipeline is at Map Manager with the python APIs being these two steps:

#4 syncMapTranslate

#5 syncMapTranslateReply

See Python API, internal for general information about python APIs as well as an example call and response.
