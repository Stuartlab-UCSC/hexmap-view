Place New NodesÂ¶

Place your nodes on the map by clicking on the Edit menu then clicking on Place New Nodes.

The resulting window, Place New Nodes is described below.

Not all maps have the information required to place new nodes. If this option is greyed out in the Edit menu, then this map does not have the required information.

If this option does not exist in the Edit menu, then you do not have the proper authority to run this compute-intensive task. Ask hexmap at ucsc dot edu with justification for authorization.

clustering data : Select a local file to upload that contains the data for your nodes. The file needs to be in tab-separated-values format as described in the window.
