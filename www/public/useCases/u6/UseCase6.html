Objective:

Identify genomic events that distinguish one subtype of cancer (e.g. basal breast cancer) from another (e.g. luminal breast cancer).

Skills Covered:

  • Creating a new group from an existing categorical (multi-valued) attribute.
  • Using set operations to combine existing groups.
  • Running statistical test to order attributes based on their differential presence/absence in one group versus another.

Steps:

  1. In the Tumor Map, make sure the mRNA layout is selected
  2. In the Short List, make sure Tissue is checked and no other attributes are checked

  1. Observe that breast tumors are in blue on the bottom right of the map
  2. In the Select Attribute type in “brca” and click on BRCA Subtype

  1. Observe the the breast tumors are colored by the molecular subtypes
  2. Uncheck Tissue attribute in the Short List

  1. Under the BRCA Subtype attribute click on the filter checkbox and select Basal in the drop down
  2. Click Save Filter next to the filter drop down

  1. Specify the name of the group (e.g. BasalBRCA) in the popup box
  2. Under the BRCA Subtype attribute click on the filter checkbox and select LumA in the drop down
  3. Click Save Filter next to the filter drop down

  1. Specify the name of the group (e.g. LumABRCA) in the popup box
  2. Under the BRCA Subtype attribute click on the filter checkbox and select LumB in the drop down
  3. Click Save Filter next to the filter drop down

  1. Specify the name of the group (e.g. LumBBRCA) in the popup box
  2. Click on Set Operations next to the Short List
  3. Select Union in the drop down list
  4. Select LumABRCA as the first attribute
  5. Select LumBBRCA as the second attribute
  6. Click on Compute Set Operation

  1. Specify the name of the new group (e.g. LumBRCA) in the popup box
  2. Click on the Sort Attributes button (next to Sorted By label)
  3. In the popup click on Differential attributes radio button
  4. Select BasalBRCA as the first attribute
  5. Select LumBRCA as the second attribute
  6. Click Sort

  1. Wait until Computing statistics now … disappears
  2. Observe that the top differentiating attributes in the Select Attribute drop down list are basal gene program and estrogen gene program (see the illustration)
